## **Replies Setting**

Configure how Fini will handle replies
![Step 4 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/1fdf1a43-8256-4f60-b4a5-59b2e1ab55ec/45f207fd-ea48-4b3b-9f96-443261e1a2b5.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200)


<Steps>
  <Step title="Select Time Delay">
  - To make Fini's response time feel more human-like, you can set a delay in seconds. For instance, 1 hour delay would be set as `3600` seconds. But for Chat we recommend to set it as 0.
     ![Step 3 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/3798f96a-95a3-4106-a3b3-3b4f3be7d95f/15771843-9010-4334-a5b1-e5a80c1d2337.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=610&mark-y=114&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTcmaD02NiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

  </Step>
<Step title="Set a Debounce Lag">
     - This feature is useful for chat scenarios where the user sends information in multiple messages. Debounce lag allows the bot to wait for a specified number of seconds before responding. During this time, Fini assesses all incoming messages to generate a more relevant response based on complete understanding of the user issue.
    ![Step 4 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/67911aca-00ce-4a2a-bd82-0900396974d4/c5fddba7-8660-4fc6-8f8d-7e8e732684f3.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=607&mark-y=226&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTAmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw
  </Step>
<Step title="Choose a Reply Type">
    Fini supports two types of replies:
  - **Direct Replies**: Direct responses to customers.
  - **Internal Comments**: Internal notes visible only to your support agents.

![Step 5 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/8f5eee1c-a62a-49b4-89e5-54dcccac300a/e964eeb7-7983-48ac-b04e-4ba84d3d330c.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=610&mark-y=313&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTUmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Reply Type if Agent Assigned">
    Configure how Fini behaves when a support agent is assigned to a ticket:
  - **Reply to End User**: Fini continues engaging with the customer even after agent assignment.
  - **Post Internal Comment**: Fini stops communicating directly with the customer and instead posts updates as internal comments for the support team.
  - **No Reply**: Fini stops all activity on the ticket, allowing the agent to handle it entirely.

![Step 6 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/ea3d204b-2f09-4b26-9c63-f01faba79d60/a454014a-45bc-455c-9321-b21b0fd5aec6.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=606&mark-y=402&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTUmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


  </Step>
<Step title="PII Masking">
    To protect **Personally Identifiable Information (PII)**, enable PII masking. When enabled, Fini will avoid using personal data (like the user's name, email etc.) in responses, providing a generalized interaction.

**Example**:  
With PII masking:  
*"Hello, how can I assist you today?"*

Without PII masking:  
*"Hello, John, how can I assist you today?"*

![Step 7 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/0f2cac8d-14a5-4d52-bee7-6c1984b20791/6474b042-e967-4e6f-a7ec-50e4be67af9e.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=605&mark-y=504&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTUmaD02NiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
</Steps>

##  **Escalation Setting**

These settings determine how Fini responds when an issue is escalated to a human agent

![Step 10 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/6a9ba142-1a26-4e30-8e45-ac7772439ad0/72ba2af2-fd57-4254-bcbc-9389c33cf459.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=30&mark-y=19&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz0yMjgmaD00MiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


<Steps>
  <Step title="Action if Escalated">
    Define what happens if Fini encounters a question it cannot answer (e.g., sensitive topics):
    - **Continue Responding**: Fini will continue to engage even after escalation.
    - **No Reply**: Fini stops replying after escalation.
    - **Send Acknowledgement Message**: Fini sends a customizable acknowledgment message to the customer once escalation occurs.

    **Suggested Message**
    > "Thank you for the additional information. This will be included in the ticket for our specialized human agent team."


    ![Step 9 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/ed8894f1-4e0b-47b0-8f27-0087b3f94422/94122291-ec17-4c07-a2b7-bc7ae6a701bd.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=610&mark-y=123&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NDYmaD02NCZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Acknowledgment Message">
     - Customize the message users receive when their query is escalated. 

    **Suggested Message**:  
    > "Thank you for the additional information. This will be included in the ticket for our specialized human agent team."

      ![Step 10 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/c1639cc4-41ba-45f7-a4f0-13056e737377/f5ee71e5-e3c6-4fd2-8a32-bc3474536910.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=602&mark-y=205&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NDkmaD0xNDYmZml0PWNyb3AmY29ybmVyLXJhZGl1cz0xMA%3D%3D)
    
  </Step>
<Step title="Threshold Count for Escalation">
     - Set a **threshold** for the maximum number of replies before escalation. If Fini cannot resolve an issue within a set number of replies (for chat we recommend to put it as high as 100), the ticket is escalated to a human agent.


    ![Step 11 screenshot](https://images.tango.us/workflows/6094487b-e17a-4cc5-acfa-87525f1539c1/steps/b4ea9eee-8af9-4eb2-a318-0b926ba65344/ef9eed2a-d5d9-4bd7-b967-677c2858b614.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=599&mark-y=376&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NDgmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Message When Escalated via Threshold">
    - Customize the message users receive when the bot reaches the escalation threshold. 

    **Suggested Message When Threshold is Reached**:  
    > "Thank you for your message. Unfortunately, this is outside of my expertise, and I will need to escalate this to our specialized support team. A member of our team will reach out shortly."


    ![Step 2 screenshot](https://images.tango.us/workflows/5628193b-39ec-46e9-82f5-ab17d61a8baa/steps/ba6978a2-1959-45b6-a004-be2d69217797/9810f1e1-ef3d-4c36-98c5-240caa9ac8f6.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=597&mark-y=476&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTImaD0xODEmZml0PWNyb3AmY29ybmVyLXJhZGl1cz0xMA%3D%3D)
  </Step>
</Steps>

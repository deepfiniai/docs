## **Replies Setting**

Configure how Fini will handle replies
![Step 4 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/1fdf1a43-8256-4f60-b4a5-59b2e1ab55ec/45f207fd-ea48-4b3b-9f96-443261e1a2b5.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200)


<Steps>
  <Step title="Select Time Delay">
  - To make Fini's response time feel more human-like, you can set a delay in seconds. For instance, 1 hour delay would be set as `3600` seconds. But for Chat we recommend to set it as 0.
    ![Step 2 screenshot](https://images.tango.us/workflows/6e2d33b0-f769-4917-ad3c-28b34d4c2055/steps/9f19b899-0a24-4c17-89e7-441a6bc60b31/2cb249e2-d65b-4bd8-8d44-e02be0184ed7.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=601&mark-y=129&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MzImaD02MyZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

  </Step>
<Step title="Set a Debounce Lag">
     - This feature is useful for chat scenarios where the user sends information in multiple messages. Debounce lag allows the bot to wait for a specified number of seconds before responding. During this time, Fini assesses all incoming messages to generate a more relevant response based on complete understanding of the user issue.
    ![Step 4 screenshot](https://images.tango.us/workflows/6e2d33b0-f769-4917-ad3c-28b34d4c2055/steps/b009f559-1c16-4ddb-94a2-e8728330f988/990c456e-6d27-4da4-b71c-4f0fdeb30fe1.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=604&mark-y=232&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTUmaD02NiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Choose a Reply Type">
    Fini supports two types of replies:
  - **Direct Replies**: Direct responses to customers.
  - **Internal Comments**: Internal notes visible only to your support agents.

![Step 5 screenshot](https://images.tango.us/workflows/6e2d33b0-f769-4917-ad3c-28b34d4c2055/steps/b3fe845a-3c99-4bb1-8d01-0e336044311f/e5d5b777-989d-4f64-8c48-ec709d823837.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=606&mark-y=317&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTEmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Reply Type if Agent Assigned">
    Configure how Fini behaves when a support agent is assigned to a ticket:
  - **Reply to End User**: Fini continues engaging with the customer even after agent assignment.
  - **Post Internal Comment**: Fini stops communicating directly with the customer and instead posts updates as internal comments for the support team.
  - **No Reply**: Fini stops all activity on the ticket, allowing the agent to handle it entirely.

![Step 6 screenshot](https://images.tango.us/workflows/6e2d33b0-f769-4917-ad3c-28b34d4c2055/steps/628918aa-5851-445d-9a85-80b3b61e860e/74c968e5-418f-4589-b3f1-58dda4d7b26c.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=604&mark-y=415&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTAmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


  </Step>
<Step title="PII Masking">
    To protect **Personally Identifiable Information (PII)**, enable PII masking. When enabled, Fini will avoid using personal data (like the user's name, email etc.) in responses, providing a generalized interaction.

**Example**:  
With PII masking:  
*"Hello, how can I assist you today?"*

Without PII masking:  
*"Hello, John, how can I assist you today?"*

![Step 7 screenshot](https://images.tango.us/workflows/6e2d33b0-f769-4917-ad3c-28b34d4c2055/steps/304b9111-c35c-47b1-8cfb-d885b70269bc/ceba87e0-ede5-4f54-918f-cb2a28e14878.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=608&mark-y=516&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTImaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
</Steps>

##  **Escalation Setting**

These settings determine how Fini responds when an issue is escalated to a human agent

![Step 10 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/6a9ba142-1a26-4e30-8e45-ac7772439ad0/72ba2af2-fd57-4254-bcbc-9389c33cf459.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=30&mark-y=19&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz0yMjgmaD00MiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


<Steps>
  <Step title="Action if Escalated">
    Define what happens if Fini encounters a question it cannot answer (e.g., sensitive topics):
    - **Continue Responding**: Fini will continue to engage even after escalation.
    - **No Reply**: Fini stops replying after escalation.
    - **Send Acknowledgement Message**: Fini sends a customizable acknowledgment message to the customer once escalation occurs.

    **Suggested Message**
    > "Thank you for the additional information. This will be included in the ticket for our specialized human agent team."


    ![Step 2 screenshot](https://images.tango.us/workflows/1e4baf57-fdc4-40b5-99b7-e1984edcb6b8/steps/65b5c3bf-a617-4309-b3fb-1cc943383e4c/ebde1d3b-83f1-4d8d-a325-b61f7de050a6.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=603&mark-y=119&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NDgmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Acknowledgment Message">
     - Customize the message users receive when their query is escalated. 

    **Suggested Message**:  
    > "Thank you for the additional information. This will be included in the ticket for our specialized human agent team."

    ![Step 3 screenshot](https://images.tango.us/workflows/1e4baf57-fdc4-40b5-99b7-e1984edcb6b8/steps/5eea83b3-18e7-4267-91ce-faa0dbba9c3f/87ffa663-46f8-436a-a0ce-f68041f9df36.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=611&mark-y=204&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTQmaD0xMjcmZml0PWNyb3AmY29ybmVyLXJhZGl1cz0xMA%3D%3D)
  </Step>
<Step title="Threshold Count for Escalation">
     - Set a **threshold** for the maximum number of replies before escalation. If Fini cannot resolve an issue within a set number of replies (for chat we recommend to put it as high as 100), the ticket is escalated to a human agent.


    ![Step 4 screenshot](https://images.tango.us/workflows/1e4baf57-fdc4-40b5-99b7-e1984edcb6b8/steps/0a1c2b57-653b-47d7-85b3-bf078f1f48bf/09366f3c-4bf1-4566-b015-04786365df48.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=618&mark-y=369&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NzAmaD02NyZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Message When Escalated via Threshold">
    - Customize the message users receive when the bot reaches the escalation threshold. 

    **Suggested Message When Threshold is Reached**:  
    > "Thank you for your message. Unfortunately, this is outside of my expertise, and I will need to escalate this to our specialized support team. A member of our team will reach out shortly."


    ![Step 5 screenshot](https://images.tango.us/workflows/1e4baf57-fdc4-40b5-99b7-e1984edcb6b8/steps/6f9d8ded-24ef-400a-abda-8f925acf5078/3113cefa-bb03-4222-927a-90c5fb932712.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=617&mark-y=452&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NjImaD0xODImZml0PWNyb3AmY29ybmVyLXJhZGl1cz0xMA%3D%3D)
  </Step>
</Steps>

    This field has to already exists in HubSpot and contain the required categories
![Step 23 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/9b6d311c-c015-42b5-a50c-bbdb9b6c6a7e/b2e4835e-7524-44d0-bfc0-6d5d9211ab85.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=591&mark-y=217&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MjImaD02MiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

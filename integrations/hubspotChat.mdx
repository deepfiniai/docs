## 2. Replies Setting

Configure how Fini will handle replies
![Step 4 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/1fdf1a43-8256-4f60-b4a5-59b2e1ab55ec/45f207fd-ea48-4b3b-9f96-443261e1a2b5.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200)


#### **1. Select Time Delay**
- To make Fini's response time feel more human-like, you can set a delay in seconds. For instance, 1 hour delay would be set as `3600` seconds. But for Chat we recommend to set it as 0.

![Step 1 screenshot](https://images.tango.us/workflows/57d6bd03-554f-4e6e-a112-62bf0b24d98a/steps/8c275c66-b389-4a49-b814-47e6ef1ef6e7/caab2c5d-3eab-4354-861f-d198891d7e2a.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&border=2%2CF4F2F7&mark-x=600&mark-y=121&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NDYmaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


#### **2. Set a Debounce Lag**
- This feature is useful for chat scenarios where the user sends multiple messages on the same issue. The bot will wait for a specified number of seconds before responding, allowing it to update its reply based on new information.

![Step 2 screenshot](https://images.tango.us/workflows/57d6bd03-554f-4e6e-a112-62bf0b24d98a/steps/e284b267-6173-438f-9ceb-16dac63daa7f/6c121112-2e44-409d-abba-994806f4a1ba.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&border=2%2CF4F2F7&mark-x=587&mark-y=227&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MzUmaD02MyZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


#### **3. Choose a Reply Type**

Fini supports two types of replies:
- **Direct Replies**: Direct responses to customers.
- **Internal Comments**: Internal notes visible only to your support agents.

![Step 1 screenshot](https://images.tango.us/workflows/0b056610-8d65-49c2-a2e0-1c863a419045/steps/7130a75b-6801-4560-b66b-3fa441b4b2bd/8a83cd2c-5d98-4a3c-b98b-6c8f4b0af4e8.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=606&mark-y=331&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NDAmaD02NCZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **4. Reply Type if Agent Assigned**

Configure how Fini behaves when a support agent is assigned to a ticket:
- **Reply to End User**: Fini continues engaging with the customer even after agent assignment.
- **Post Internal Comment**: Fini stops communicating directly with the customer and instead posts updates as internal comments for the support team.
- **No Reply**: Fini stops all activity on the ticket, allowing the agent to handle it entirely.


![Step 2 screenshot](https://images.tango.us/workflows/0b056610-8d65-49c2-a2e0-1c863a419045/steps/d7416ef7-87d7-4565-b576-02f6699cc98b/7c6082fb-c2dc-4bb0-b57f-158a9ebc7003.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=588&mark-y=414&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MjgmaD02MiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **5. PII Masking**

To protect **Personally Identifiable Information (PII)**, enable PII masking. When enabled, Fini will avoid using personal data (like the user's name, email etc.) in responses, providing a generalized interaction.

**Example**:  
With PII masking:  
*"Hello, how can I assist you today?"*

Without PII masking:  
*"Hello, John, how can I assist you today?"*

##  **3. Escalation Setting**

These settings determine how Fini behaves when an issue is escalated to a human agent

![Step 10 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/6a9ba142-1a26-4e30-8e45-ac7772439ad0/72ba2af2-fd57-4254-bcbc-9389c33cf459.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=30&mark-y=19&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz0yMjgmaD00MiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


####  **1. Action if Escalated**
 Define what happens if Fini encounters a question it cannot answer (e.g., sensitive topics):
    - **Continue Responding**: Fini will continue to engage even after escalation.
    - **No Reply**: Fini stops replying after escalation.
    - **Send Acknowledgement Message**: Fini sends a customizable acknowledgment message to the customer once escalation occurs.

**Suggested Message**
    > "Thank you for the additional information. This will be included in the ticket for our specialized human agent team."


![Step 11 screenshot](https://images.tango.us/workflows/bb5e53e9-76e4-4050-a437-cc943c6ff822/steps/70ad10c9-e91f-421d-8844-92b2b137a570/15676301-5996-43ef-8795-3658cc0aaf19.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=605&mark-y=118&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NTImaD02NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

- Customize the message users receive when their query is escalated.

#### **3. Threshold Count for Escalation**

- Set a **threshold** for the maximum number of replies before escalation. If Fini cannot resolve an issue within a set number of replies (e.g., 5 or 10), the ticket is escalated to a human agent. For Chat conversation we recommend to select a higher number (like 50 or 100). Sometimes Chat conversation can have numerous replies.

![Step 1 screenshot](https://images.tango.us/workflows/437ad3ad-98c6-4ef8-8e99-0c3c04843630/steps/080aa84c-7289-45a0-ba68-8f0f671362c3/94a2c889-7049-4073-9aa2-3e9e29f793dc.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=578&mark-y=311&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MjEmaD02MSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **4. Message When Escalated via Threshold**
- Customize the message users receive when the bot reaches the escalation threshold. 

**Suggested Message When Threshold is Reached**:

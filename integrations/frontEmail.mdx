## **Replies Setting**

![Step 6 screenshot](https://images.tango.us/workflows/359bbbef-60b4-4a37-a69f-6016dd3d0058/steps/8d0b6512-5073-4034-a6c9-df0f04893875/27ba2bf3-34e6-443d-814a-715465622244.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200)

Configure how Fini will handle replies:

#### **Set a Debounce Lag**
    - This feature is useful for chat scenarios where the user sends multiple messages on the same issue. The bot will wait for a specified number of seconds before responding, allowing it to update its reply based on new information.

![Step 7 screenshot](https://images.tango.us/workflows/359bbbef-60b4-4a37-a69f-6016dd3d0058/steps/5ded5035-a72f-4d18-baab-cc3bfacf2593/55d53bc2-48ac-4279-987f-ee4fd5c34dab.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=595&mark-y=131&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MzQmaD02MyZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **Choose a Reply Type**

Fini supports two types of replies:
- **Direct Replies**: Direct responses to customers.
- **Internal Comments**: Internal notes visible only to your support agents.

![Step 8 screenshot](https://images.tango.us/workflows/359bbbef-60b4-4a37-a69f-6016dd3d0058/steps/8bfce238-a2a2-417f-9e4a-1c4a3611145a/bcacb8b6-24fa-4139-8851-f64426b01f48.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=596&mark-y=215&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MzEmaD02MyZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **Reply Type if Agent Assigned**

Configure how Fini behaves when a support agent is assigned to a ticket:
- **Reply to End User**: Fini continues engaging with the customer even after agent assignment.
- **Post Internal Comment**: Fini stops communicating directly with the customer and instead posts updates as internal comments for the support team.
- **No Reply**: Fini stops all activity on the ticket, allowing the agent to handle it entirely.


![Step 9 screenshot](https://images.tango.us/workflows/359bbbef-60b4-4a37-a69f-6016dd3d0058/steps/9d25766f-0de3-4043-abf5-a9a6d122dd41/7fd28098-3179-4e4b-a9a1-34edde3933bc.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=589&mark-y=282&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MjQmaD02MiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


#### **PII Masking**

To protect **Personally Identifiable Information (PII)**, enable PII masking. When enabled, Fini will avoid using personal data (like the user's name, email etc.) in responses, providing a generalized interaction.

**Example**:  
With PII masking:  
*"Hello, how can I assist you today?"*

Without PII masking:  
*"Hello, John, how can I assist you today?"*

##  **Escalation Setting**

These settings determine how Fini behaves when an issue is escalated to a human agent.

![Step 1 screenshot](https://images.tango.us/workflows/2f3f2eb5-a84d-49b5-80a0-53589d64f36d/steps/f315700f-352a-4361-937c-7141a005c7e1/1cca65b7-83b5-44ce-9651-87111cfa9b9f.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200)

####  **Action if Escalated**
    Define what happens if Fini encounters a question it cannot answer (e.g., sensitive topics):
    - **Continue Responding**: Fini will continue to engage even after escalation.
    - **No Reply**: Fini stops replying after escalation.
    - **Send Acknowledgement Message**: Fini sends a customizable acknowledgment message to the customer once escalation occurs.

    **Suggested Message**
    > "Thank you for the additional information. This will be included in the ticket for our specialized human agent team."

![Step 12 screenshot](https://images.tango.us/workflows/359bbbef-60b4-4a37-a69f-6016dd3d0058/steps/cbf13d9d-2cfc-4d49-a92a-6f26f9a912ec/c3363a63-f6bc-4347-8282-44564cec8bc5.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=597&mark-y=125&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MjQmaD02MiZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **Acknowledgment Message**
    - Customize the message users receive when their query is escalated. 

    **Suggested Message**:  
    > "Thank you for the additional information. This will be included in the ticket for our specialized human agent team."

![Step 1 screenshot](https://images.tango.us/workflows/b02527e8-30cc-425c-9c11-a942316791c0/steps/9af4ac61-86d0-446e-8b1c-755d9109e735/f321cb20-96aa-4ea5-be12-7b3625b0d38a.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=586&mark-y=213&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01MjMmaD05MyZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **Threshold Count for Escalation**
    - Set a **threshold** for the maximum number of replies before escalation. If Fini cannot resolve an issue within a set number of replies (e.g., 5 or 10), the ticket is escalated to a human agent.


![Step 14 screenshot](https://images.tango.us/workflows/359bbbef-60b4-4a37-a69f-6016dd3d0058/steps/cd7cb895-f287-4b24-abe7-b6c4224b0bed/a47c7b2a-d420-4fb9-b4a5-9d4add5d6748.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=608&mark-y=342&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz01NDAmaD02NCZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)

#### **Message When Escalated via Threshold**
    - Customize the message users receive when the bot reaches the escalation threshold. 

    **Suggested Message When Threshold is Reached**:  
    > "Thank you for your message. Unfortunately, this is outside of my expertise, and I will need to escalate this to our specialized support team. A member of our team will reach out shortly."

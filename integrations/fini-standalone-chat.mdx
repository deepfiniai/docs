## General setup

<Steps>
  <Step title="Add Domain">
    You have 3 options while you select the domain: 
    1) Enter specific domain: For eg: `https://www.usefini.com` - when you do this your `Fini Standalone` will only work on the selected   website. 
    2) Enter partial domain : For eg: `*.usefini.com` - you can choose this when you want to use the `Fini Standalone` across differents parts of your product (eg: app, website etc.)
    3) Enter "null": when you want your `Fini Standalone` to work across domains you can enter "null" and then you can deploy it across domains

## Deploy Fini Standalone

There are 2 ways to use the standalone link: 
1) Generate Fini standalone link and use the link directly 
2) Deploy the link on your website

### Generate Fini standalone link

Go back to [deploy page](https://app.usefini.com/dashboard/deploy/widget). Click on *Preview* -> *Standalone*
This will open a new standalone page. You can copy the corresponding URL and share it with your team.
![Step 22 screenshot](https://images.tango.us/workflows/3b180847-f9c8-4d28-8732-4d32ec680742/steps/72010132-daec-4a07-9383-96dc54cc54e8/0cd2eacc-46fb-4beb-a098-64be93e3cf88.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=922&mark-y=183&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz0xOTEmaD00MSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)


### Deploy on website


<Steps>
  <Step title="Select Embed Script">
    ![Step 24 screenshot](https://images.tango.us/workflows/e6be1ee4-5070-41cc-9838-157fe1bb19e4/steps/194c77c8-637c-435a-97e0-efb589922a25/5cf2801d-c8c7-4d90-be0f-c2d4a12be3ae.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&mark-x=425&mark-y=73&m64=aHR0cHM6Ly9pbWFnZXMudGFuZ28udXMvc3RhdGljL2JsYW5rLnBuZz9tYXNrPWNvcm5lcnMmYm9yZGVyPTQlMkNGRjc0NDImdz0xODImaD01NSZmaXQ9Y3JvcCZjb3JuZXItcmFkaXVzPTEw)
  </Step>
<Step title="Copy code">
    Copy and paste this script to the HTML file of your website. Must put the snippet inside or after the body tag!


```html Fini-standalone.html

  <script>
      (function (w, d, s, o, f, js, fjs) {
        (js = d.createElement(s)), (fjs = d.getElementsByTagName(s)[0]);
        js.id = o;
        js.src = f;
        js.async = 1;
        fjs.parentNode.insertBefore(js, fjs);
        js = d.createElement("div");
        js.id = "fini-widget-root";
        d.body.appendChild(js);
        w.finiWidgetOptions = {
          widgetId: "yNYoJYuGMW",
          mode: "standalone" 
        };
      })(window, document, "script", "FiniWidget", "https://storage.googleapis.com/fini-widget/assets/fini-widget-prod.min.js");
  </script>
  
```
  </Step>
</Steps>

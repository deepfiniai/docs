### **What Are Fini Knowledge Items?**
Fini Knowledge Items aims to be your new single source of truth for customer support in the near future. This feature allows you to consolidate all your macros, question-answer pairs, and similar questions in one centralized location. By doing so, it ensures that specific macros are used consistently and accurately across all customer interactions.


The below guide is divided into five sections:
1. Default bots
2. Knowledge Item Creation
  - Select bot
  - Create knowledge (question, answer, instructions)
  - Keywords, categories, escalation, similar questions
  - Statuses
3. Edit Knowledge Items
4. Filter Knowledge Items
5. Delete Knowledge Items

## 1.3. Statuses

Once the Knowledge Item is ready you can move over to the status section and decide to set the status to:

1. "Live" so that Fini can start using it straight away

2. "Draft" if you know you'll have to work on it
    
3. "In review" in case there is someone who needs to approve the final version.
    

_Fini won't use a Knowledge item unless the status is set as LIVE_

### **Filter options**

You can filter knowledge items by:

1.  User (Who has created the KI)
    
2.  Status (Live, In Review, Draft)
    
3.  Number (of the KI on your account)
    
4.  Question (can be a part of the question, or just a word)
    
5.  Category
    
6.  Subcategory
    
7.  Keyword
    
8.  Escalation (yes or no)
    
9.  Answer (can be a part of the answer, or just a word)

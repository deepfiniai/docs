---
title: "Refresh links"
api: "POST https://api.usefini.com/api/v1/public/bots/refresh"
description: "This endpoint allows you to refresh links in your existing bot"
---

### Headers

<ParamField header="token" type="string" required>
  This is the unique api-key for your bot 
</ParamField>

### Body

<ParamField body="bot_links" type="string" required>
List of links to be refreshed. Example [sample_bot_link_1, sample_bot_link_2]
</ParamField>

### Response

<ResponseField name="message" type="string">
  Indicates whether the request for links refresh was sent successfully to the server.
</ResponseField>


<RequestExample>

```python Example Request- Python
import requests

url = 'https://api.usefini.com/api/v1/public/bots/links/refresh'
token = "your_api_key_here"
headers = {
    'authorization': 'Bearer token',
}

data = {
    "bot_links": [ bot_link_1, bot_link_2]
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

```javascript Example Request- Node.js
const axios = require('axios');

const url = 'https://api.usefini.com/api/v1/public/bots/links/refresh';
const token = 'your_api_key_here'; // Replace with your actual API key

const headers = {
  'authorization': `Bearer ${token}`
};

const data = {
  "bot_links": ["bot_link_1", "bot_link_2"]
};

axios.post(url, data, { headers })
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error('Error:', error);
  });
```

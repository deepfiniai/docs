---
title: "Delete links"
api: "POST https://api.usefini.com/api/v1/bots/links/delete/public"
description: "This endpoint allows you to delete links from your existing bot"
---

### Body

<ParamField body="links" type="string" required>
List of links to be deleted from your bot. Example [sample_link_1, sample_link_2]
</ParamField>

### Response

<ResponseField name="message" type="string">
  Indicates whether the URLs were successfully updated.
</ResponseField>


<RequestExample>

```python Example Request- Python
import requests

url = 'https://api.usefini.com/api/v1/bots/links/delete/public'
token = "your_api_key_here"
headers = {
    'authorization': 'Bearer token',
}

data = {
    "links": [ link_1, link_2]
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

```javascript Example Request- Node.js
const axios = require('axios');

const url = 'https://api.usefini.com/api/v1/bots/links/delete/public';
const token = 'your_api_key_here'; // Replace with your actual API key

const headers = {
  'authorization': `Bearer ${token}`
};

const data = {
  "links": ["link_1", "link_2"]
};

axios.post(url, data, { headers })
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error('Error:', error);
  });
```

</RequestExample>

---
title: "Add links"
api: "POST https://api-prod.usefini.com/v2/bots/links/public"
description: "This endpoint allows you to add links to your existing bot"
---

### Body

<ParamField body="links" type="string" required>
List of links to be added to your bot. Example [sample_link_1, sample_link_2]
</ParamField>

<ParamField body="english" type="boolean">
Source language for links. 
- If content is in english, then set this as true
- If content is in any other language, then set this as false. In this case, language will be autodetected before the bot is trained
</ParamField>

<ParamField body="baser" type="boolean">
Set this as true if you want high quality bot creation.  
</ParamField>

### Response

<ResponseField name="message" type="string">
  Indicates whether the request for links addition was sent successfully to the server.
</ResponseField>


<RequestExample>

```python Example Request- Python
import requests

url = 'https://api-prod.usefini.com/v2/bots/links/public'
token = "your_api_key_here"
headers = {
    'authorization': 'Bearer token',
}

data = {
    "links": [ link_1, link_2],
    "english": True,
    "baser": False
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

```javascript Example Request- Node.js
const axios = require('axios');

const url = 'https://api-prod.usefini.com/v2/bots/links/public';
const token = 'your_api_key_here'; // Replace with your actual API key

const headers = {
  'authorization': `Bearer ${token}`
};

const data = {
  "links": ["link_1", "link_2"],
  "english": true,
  "baser": false
};

axios.post(url, data, { headers })
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error('Error:', error);
  });
```

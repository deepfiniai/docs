---
title: "Add links"
api: "PUT https://api.usefini.com/api/v1/public/bots/links"
description: "This endpoint allows you to add links to your existing bot"
---

### Authorization

<ParamField header="token" type="string" required>
  This is the unique api-key for your bot 
</ParamField>

### Body

<ParamField body="bot_links" type="string" required>
List of links to be added to your bot. Example [sample_bot_link_1, sample_bot_link_2]
</ParamField>

<ParamField body="english" type="boolean">
Links source language. If the links content is in english then set this as true, othwerwise false. We will autodetect the language if it set as false. 
</ParamField>

<ParamField body="baser" type="boolean">
Set this as true if you want high quality bot creation.  
</ParamField>

<ParamField body="isParentUrl" type="boolean">
If the URL is a parent URL then set this as true.
</ParamField>


### Response

<ResponseField name="message" type="string">
  Indicates whether the request for links addition was sent successfully to the server.
</ResponseField>


<RequestExample>

```python Example Request- Python
import requests

url = 'https://api.usefini.com/api/v1/public/bots/links'
token = "your_api_key_here"
headers = {
    'authorization': 'Bearer token',
}

data = {
    "bot_links": [ bot_link_1, bot_link_2],
    "english": True,
    "baser": False,
    "isParentUrl": False
}

response = requests.put(url, json=data, headers=headers)
print(response.json())
```

```javascript Example Request- Node.js
const axios = require('axios');

const url = 'https://api.usefini.com/api/v1/public/bots/links';
const token = 'your_api_key_here'; // Replace with your actual API key

const headers = {
  'authorization': `Bearer ${token}`
};

const data = {
  "bot_links": ["bot_link_1", "bot_link_2"],
  "english": true,
  "baser": false,
  "isParentUrl": false
};

axios.put(url, data, { headers })
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error('Error:', error);
  });
```

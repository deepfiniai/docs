---
title: "Store feedback"
api: "POST https://api-prod.usefini.com/v2/bot-requests/feedback"
description: "This endpoint allows you to store feedback from answers in the /answer endpoint"
---

### Body

<ParamField body="answerId" type="string" required>
 This is the anwer_uuid retrieved from /answer response object
</ParamField>

<ParamField body="feedbackType" type="string" required>
  This is the feedback type from users. It can be set to [“thumbs_up”, “thumbs_down”]
</ParamField>


### Response

<ResponseField name="message" type="string">
  Indicates whether the feedback was added.
</ResponseField>


<RequestExample>

```python Example Request- Python
import requests
endpoint = "https://api-prod.usefini.com/v2/bot-requests/feedback"

answerId = <uuid> # Replace this with the retrieved answer_uuid from /answer endpoint
feedbackType = "thumbs_up" # or "thumbs_down"
data = {“answerId”: answerId, “feedbackType”: feedbackType}

token = "your_api_key_here"  # Best stored in secrets management
headers = {"Authorization": "Bearer " + token}

# Make sure your ‘content-type’ header is set to ‘application/json’
response = requests.post(endpoint, json=data, headers=headers)
print(response.json());
```

```javascript Example Request- Node.js
const axios = require('axios');  // Make sure to install axios: npm install axios

const endpoint = 'https://api-prod.usefini.com/v2/bot-requests/feedback';

const answerId = <uuid>; // Replace this with the retrieved answer_uuid from /answer endpoint
const feedbackType = "thumbs_up" //or "thumbs_down";

const data = { answerId: answerId, feedbackType: feedbackType }; 

const token = 'your_api_key_here'; // Best stored in secrets management

const headers = {
    Authorization: 'Bearer ' + token,
    'Content-Type': 'application/json'
};

axios.post(endpoint, data, { headers: headers })
    .then((response) => {
        console.log(response.data);
    })
    .catch((error) => {
        console.error(error);
    });
```

</RequestExample>

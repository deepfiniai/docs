---
title: "Store feedback"
api: "POST https://api.usefini.com/api/v1/feedback"
description: "This endpoint allows you to store feedback from answers in the /answer endpoint"
---

### Body

<ParamField body="token" type="string" required="true">
  This is the unique api-key for your bot 
</ParamField>

<ParamField body="answer_uuid" type="string" required="true">
 This is the anwer_uuid retrieved from /answer response object
</ParamField>

<ParamField body="feedback_type" type="string" required="true">
  This is the feedback type from users. It can be set to [“thumbs_up”, “thumbs_down”]
</ParamField>


### Response

  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<ResponseField name="based_on" type="object">

 Link of the source that was used to generate the answer

Matching score with the user query
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="messages" type="object">

A list of messages comprising the conversation so far.

<Expandable title="Toggle object">

<ResponseField name="role" type="string">
  The role of the messages author. One of `system`, `user`, `assistant`, or `function`.
</ResponseField>

<ResponseField name="content" type="string">
 The contents of the message. `content` is required for all messages except assistant messages with function calls.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the author of this message. `name` is required if role is `function`, and it should be the name of the function whose response is in the `content`. May contain a-z, A-Z, 0-9, and underscores, with a maximum length of 64 characters.
</ResponseField>

The name and arguments of a function that should be called, as generated by the model.

```python Example Request- Python
import requests
endpoint = "https://api.usefini.com/api/v1/feedback"

answer_uuid = <uuid> # Replace this with the retrieved answer_uuid from /answer endpoint
feedback_type = “thumbs_up” or “thumbs_down”
data = {“answer_uuid”: answer_uuid, “feedback_type”: feedback_type}

token = "your_api_key_here"  # Best stored in secrets management
headers = {"Authorization": "Bearer " + token}

# Make sure your ‘content-type’ header is set to ‘application/json’
response = requests.post(endpoint, json=data, headers=headers)
print(response.json());
```

```javascript Example Request- Node.js
const axios = require('axios');  // Make sure to install axios: npm install axios

const endpoint = 'https://api.usefini.com/api/v1/feedback';

const answer_uuid = <uuid> # Replace this with the retrieved answer_uuid from /answer endpoint;
const feedback_type = “thumbs_up” or “thumbs_down”;

const data = { answer_uuid: answer_uuid, feedback_type: feedback_type }; 

const token = 'your_api_key_here'; // Best stored in secrets management

const headers = {
    Authorization: 'Bearer ' + token,
    'Content-Type': 'application/json'
};

axios.post(endpoint, data, { headers: headers })
    .then((response) => {
        // This response will contain an updated message_history and an answer.
        // Re-use this message_history in a new request to enable chat mode
        console.log(response.data);
    })
    .catch((error) => {
        console.error(error);
    });
```

Use Fini's UI to generate the unique API key for your bot

<Note>Every bot has a unique key. If you are deploying multiple bots, create separate keys for each bot</Note>

## API Authentication

Fini verifies your identity using your secret keys. You can use your api-key to authenciate. For example:


```bash
'Authorization': 'Bearer <your_api_key_here>'
```
